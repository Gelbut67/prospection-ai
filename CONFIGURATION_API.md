# 🔑 Système de découverte de prospects réels

## 🎯 Système 100% GRATUIT - IA Hybride

**Aucune API payante requise !** ✅

---

## 📊 Fonctionnement

### **Double IA Groq (100% gratuit)**

1. **Première IA** : Génère des entreprises réelles connues
   - Basée sur les critères de recherche
   - Utilise sa connaissance d'entreprises françaises établies
   - Privilégie les entreprises avec présence web

2. **Seconde IA** : Enrichit chaque entreprise
   - Site web officiel
   - Email de contact
   - Nom du dirigeant
   - Vérification de l'existence

## 🚀 Déploiement

**Groq API déjà configurée** ✅
